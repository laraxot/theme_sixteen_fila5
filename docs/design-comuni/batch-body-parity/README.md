# Batch Body Parity Artifacts

This folder contains per-page HTML artifacts generated by the batch structural audit.

For each page slug there is a dedicated subfolder with:
- `reference-body.html`
- `local-body.html`

Use these files to inspect the HTML structure inside `<body>` without scripts before deciding whether the page is ready for CSS/JS parity work.
