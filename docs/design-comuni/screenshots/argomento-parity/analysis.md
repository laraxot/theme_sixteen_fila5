# Argomento Screenshot Analysis

## Visual Summary

La differenza visiva principale e' gia' spiegata dalla struttura.

Reference:

- hero fotografico full-width
- card bianca sovrapposta
- colonna destra con due card gestionali
- sezioni multiple sotto la piega

Locale:

- hero piatto su fondo bianco
- nessuna immagine hero
- nessuna colonna destra gestionale
- contenuto molto piu' corto

## Consequence

Il lavoro CSS-only non puo' chiudere questo gap senza prima riallineare l'HTML generato dai blocchi CMS.
