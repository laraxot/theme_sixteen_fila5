## 🔧 Integration Workflow

### Complete Workflow

```
1. BMAD Analysis
   ↓
   @bmad-analyze --target="homepage"
   ↓
   Output: specs.md

2. GSD Execution
   ↓
   @gsd-execute --task="create-block-views" --specs="specs.md"
   ↓
   Output: block-views/

3. Ralph Iteration
   ↓
   @ralph-loop --task="homepage-refinement" --iterations="5"
   ↓
   Output: refined-block-views/

4. Superpowers Enhancement
   ↓
   @superpowers enhance --task="performance-optimization"
   ↓
   Output: optimized-block-views/

5. OpenViking Context
   ↓
   openviking add-memory --title="Homepage" --file="optimized-block-views/"
   ↓
   Output: context stored

6. NotebookLM Knowledge
   ↓
   notebooklm-mcp query --source="homepage" --question="Next steps?"
   ↓
   Output: recommendations
```
