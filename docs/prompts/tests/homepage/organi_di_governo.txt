## 🔍 ROOT CAUSE ANALYSIS

### Blade Structure (CORRETTA ✅)
```blade
<div class="row g-4 mb-5">
  @foreach($cards as $i => $card)
  <div class="col-md-6 col-lg-4">  ← Bootstrap grid CORRETTA
    <div class="card ...">...</div>
  </div>
  @endforeach
</div>
```

### CSS Problem (SBAGLIATO ❌)
```css
/* design-comuni-global.css riga 17 */
.card-teaser-wrapper-equal {
  display: flex !important;
  flex-direction: row !important;
  flex-wrap: nowrap !important;  ← PROBLEMA! Force tutti in UNA riga
  gap: 1.5rem !important;
}

.card-teaser-wrapper-equal > .card {
  flex: 1 1 0 !important;  ← Ogni card occupa spazio uguale
  min-width: 0 !important;
  max-width: none !important;
}
```

### Conflitto:
- Blade usa Bootstrap grid: `row` + `col-md-6 col-lg-4` → 3 colonne
- CSS override: `flex-wrap: nowrap` → tutte in UNA riga orizzontale
- Risultato: card strettissime, occupano poco spazio orizzontale

---

## ✅ SOLUZIONE

### Opzione 1: Rimuovere classe CSS dal wrapper governance (PREFERITA)

**File**: `laravel/Themes/Sixteen/resources/views/components/blocks/governance/cards.blade.php`

Il wrapper `.card-teaser-wrapper-equal` NON deve essere usato perché:
- È progettato per flexbox layout SENZA Bootstrap grid
- Governance cards USA già Bootstrap grid (`col-md-6 col-lg-4`)

**Fix**: Rimuovere classi flexbox dal wrapper, usare solo Bootstrap row/col

### Opzione 2: CSS override per governance section

Aggiungere CSS specifico per governance cards:

```css
/* Governance cards - use Bootstrap grid, NOT flexbox */
#calendario .row.g-4 {
  display: flex !important;
  flex-wrap: wrap !important;  ← Allow wrapping!
  gap: 1rem;
}

#calendario .col-md-6.col-lg-4 {
  flex: 0 0 auto !important;  ← Don't stretch
  width: 33.333% !important;  ← 3 columns on lg
}

@media (max-width: 991.98px) {
  #calendario .col-md-6.col-lg-4 {
    width: 50% !important;  ← 2 columns on md
  }
}

@media (max-width: 575.98px) {
  #calendario .col-md-6.col-lg-4 {
    width: 100% !important;  ← 1 column on mobile
  }
}
```
