## 🎯 GOAL

Replica ogni pagina di Design Comuni rendendo:

- **URL Locale**: `http://127.0.0.1:8000/it/tests/<pagina>`

### ⚠️ Vincolo assoluto:

- **NO Bootstrap Italia**
- **SI Tailwind + Alpine.js**

---

## 🧠 PRINCIPI CORE

### DRY
- UNA sola blade: `[slug].blade.php`
- Contenuti SOLO da JSON
- Blocchi riutilizzabili

### KISS
- La blade gestisce SOLO struttura
- Il JSON gestisce SOLO contenuto
- CSS/JS fanno tutto il resto

### HTML FIRST
- Il `<body>` deve matchare ≥90% della reference
- Niente refactor finché l'HTML non è corretto

### CSS LAST
- Si lavora su CSS/JS SOLO dopo HTML corretto

🔁 WORKFLOW (LOOP UNICO)

Per ogni <pagina>:

1. ANALISI
Confronta:
Reference HTML
Local HTML (View Source)
Ignora script
Focalizzati su:
struttura DOM
gerarchia
classi
2. SCREENSHOT
Fai 3 screenshot:
originale
locale
confronto
Salva in:
docs/design-comuni/screenshots/<pagina>/
3. DIFF HTML

Documenta SOLO differenze strutturali:

Esempio:

header wrapper mancante
container annidato diverso
classi mancanti (it-*)
ordine nodi diverso

👉 NON parlare di colori o stile qui

4. FIX HTML (PRIMA DI TUTTO)
Modifica SOLO:
blade
componenti blocchi

Obiettivo:
✔ stessa struttura DOM
✔ stessi wrapper
✔ stessi livelli

5. FIX CSS/JS (DOPO HTML OK)

Lavora SOLO in:

Tailwind (@apply)
Alpine.js

Replica:

colori
spacing
layout
interazioni
6. BUILD
cd laravel/Themes/Sixteen
npm run build
npm run copy
php artisan optimize:clear
7. VALIDAZIONE

Checklist:

HTML match ≥90%
Visual match (desktop)
Responsive:
mobile (375)
tablet (768)
No testo hardcoded
No Bootstrap
8. DOCUMENTAZIONE

docs/design-comuni/
