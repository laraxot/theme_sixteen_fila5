## üîó Collegamenti Rapidi per Sviluppatori

### Setup Sviluppo
```bash
# Installazione dipendenze
cd /var/www/html/_bases/base_predict_fila3_mono/laravel/Themes/Sixteen
npm install

# Build sviluppo
npm run dev

# Build produzione
npm run build

# Copia asset
npm run copy
```

### Regole Critiche (DA RICORDARE SEMPRE)
1. **Vite**: Usare sempre `@vite([...], 'themes/Sixteen')`
2. **Layout**: Usare `<x-layouts.guest>` (gi√† AGID compliant)
3. **Route**: Usare `route('pages.view', ['slug' => 'nome'])`
4. **Componenti**: Verificare esistenza prima dell'uso
5. **Icone**: Usare `<x-filament::icon name="heroicon-o-name">`

### File di Configurazione Principali
- `vite.config.js` - Configurazione build
- `tailwind.config.js` - Configurazione Tailwind
- `package.json` - Dipendenze e script
- `theme.json` - Metadati tema
