# no docs archive rule

## Regola tema Sixteen

Per nuova documentazione del tema non usare `docs/archived/`.

- usare `docs/wiki/` per conoscenza consolidata
- usare `docs/raw/` per sorgenti di lavoro

Le directory archive legacy sono storiche e non vanno alimentate.
