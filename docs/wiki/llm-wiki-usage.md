# LLM Wiki - Usage Guide (Theme: Sixteen)

Theme docs should include responsive guidelines and component source-of-truth notes.

Suggested structure under theme docs/
- docs/raw/: design files, screenshots, CSS notes
- docs/wiki/: mobile-first component guidance, breakpoint examples, screenshots

Responsive checks:
- Add example screenshots for <=480px, 481-768px, >=769px in docs/raw/screenshots
- Add docs/wiki/responsive-checklist.md describing mobile-first checks and common fixes

QMD and indexing: same commands as modules, create collection for theme docs.
